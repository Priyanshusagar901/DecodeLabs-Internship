# Data Cleaning and Formatting Project

## Overview
This project focuses on cleaning and formatting a raw dataset to improve data quality and prepare it for further analysis. The dataset contained issues such as duplicate records, missing values, inconsistent formatting, and incorrect data types that could affect analytical results.

## Objective
The main objective of this project was to transform raw data into a clean, consistent, and analysis-ready dataset by applying essential data cleaning techniques.

## Data Cleaning Tasks Performed

### 1. Duplicate Removal
- Identified duplicate records in the dataset.
- Removed redundant entries to ensure data accuracy and consistency.

### 2. Handling Missing Values
- Detected missing or null values.
- Applied appropriate techniques to handle missing data and improve dataset completeness.

### 3. Data Formatting
- Standardized text formatting.
- Corrected inconsistent values and naming conventions.
- Improved overall dataset structure and readability.

### 4. Data Type Corrections
- Converted columns to appropriate data types.
- Ensured numerical, categorical, and date fields were stored in the correct format.
- Improved data consistency for future analysis.

## Outcome
After performing data cleaning and formatting:
- Dataset became more accurate and reliable.
